# Bookstore Frontend

This is the frontend of the Bookstore application, built using **React** with **Vite**.

## 📁 Project Structure

- `myapp/` – Contains all frontend code using React and Vite.

## 🚀 Features

- Home page with bookstore intro, signup and login options
- Register and Login using JWT authentication
- Book list fetched from Google Books API
- User profile page with API-based data fetch (user & order details)
- Cart system with quantity & pricing logic
- Checkout with shipping info, payment method, and downloadable invoice

## 🛠️ Setup Instructions
- Open a terminal, navigate to each folder, and install required packages:
    In myapp: npm install

- Navigate to the myapp folder in a new terminal.
- Start the development server using: npm run dev
- A local URL (e.g., http://localhost:5173) will appear. Open it in a browser like Chrome.
- Use the application
